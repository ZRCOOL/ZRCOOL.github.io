5. Paste the following inside the file .graphqlrc.yml

    • app:
        schema:
http://192.168.99.100:4466/

      • schema: prisma/prisma.yml
      • extensions:
        • codegen:
          • generates: ./src/generated/prisma.graphql:
            • plugins:
                - schema-ast
